# zenn-contents

https://zenn.dev/kot149

## Setup

```sh
bun install
```

Install [textlint VSCode extension](https://marketplace.visualstudio.com/items?itemName=3w36zj6.textlint)

## Creating a new article

```sh
bun new
```

Or with slug specified:

```sh
bun new --slug my-awesome-article
```

## Preview an article

```sh
bun dev
```
